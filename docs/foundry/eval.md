# City Pop — Foundry Evaluation

Stable questions that should be answerable from this wiki once it exists. Re-run on every `/foundry-eval`. Thin/missing findings feed back into the wiki index as Open Questions and Candidates.

See `eval-usage.md` for how this document is actually used.

> **Note:** as of 2026-05-07, `/root/city-pop` is a SvelteKit web app, not a wiki. If a City Pop Foundry vault gets scaffolded later, copy this file into it and start running `/foundry-eval` against it.

**Ratings:**
- `strong` — wiki has the concepts and they cite ≥2 sources (ideally mixed tiers)
- `thin` — wiki has the concepts but sourcing is weak or single-source
- `missing` — wiki can't answer

Questions are grouped by what they probe. Don't reorder or renumber — that breaks comparability across runs. Add new questions at the bottom.

**Q1. What is City Pop as a genre, and what distinguishes it from AOR, yacht rock, boogie, J-fusion, and contemporary J-pop?**
